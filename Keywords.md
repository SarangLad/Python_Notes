# Keywords of python programming

### Conditional Statements

1. ``if``
Check a condition

2. ``elif``
Else if condition

3. ``else``
else condition

4. ``match``
Pattern matching (similiar to switch)

5. ``case``
a branch inside match

### Looping Keyords
6. ``for``
for loop

7. ``while``
while loop

8. ``break``
break statement

9. ``continue``
continue statement

10. ``pass``
Do nothing place holder

### Function Keyords
11. ``def``
Define a function

12. ``return``
Return a value

13. ``lamda``
Create an anonymous function

14. ``yield``
Return values one by one (generator)

### Class & Object Oriented Programming (OOP)
15. ``class``
Define a class

16. ``self``
Refers to the current object (not a keyword, but commonly used)

### Exceptional Handling
17. ``try``
Start protected code

18. ``except``
Handle an exception

19. ``finally``
Always execute

20. ``raise``
Throw an exception

21. ``assert``
Check assumptions during debugging

### Logical Operator
22. ``and``
Logical AND

23. ``or``
Logical OR

24. ``not``
Logical NOT

### Comparison & Membership
25. ``is``
Check object identity

26. ``in``
Check whether an item exists in a collection

### Import Keywords
27. ``import``
Import a module

28. ``from``
Import specific items from a module

29. ``as``
Give an alias

Example: 
```
from math import sqrt as square_root
```
### Variable Scope
30. ``global``
Use a global variable

31. ``nonlocal``
Use a variable from the enclosing function

### Asynchronous Programming
32. ``async``
Define an asynchronous function

33. ``await``
Wait for an asynchronous operation

### Context Management
34. ``with``
Open and close resources automatically

### Boolean & Special Values
35. ``True``
Boolean true

36. ``False``
Boolean false

37. ``None``
No value or null


### Deletion
35. ``del``
Delete an object or variable
